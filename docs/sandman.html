model Module¶

The model module is repsonsible exposes the sandman.Resource class, from which user models should derive. It also makes the register() function available, which maps endpoints to their associated classes.

sandman.sandman.patch_resource(collection, lookup_id)[source]¶

Return the appropriate Response based on action performed by HTTP PATCH request.

If no resource currently exists at /<collection>/<lookup_id>, create it with lookup_id as its primary key and return a resource_created_response().

If a resource does exist at /<collection>/<lookup_id>, update it with the data sent in the request and return a no_content_response().

Note: HTTP PATCH (and, thus, patch_resource()) is idempotent

Parameters:
  • collection (string) – a sandman.model.Resource endpoint
  • lookup_id (string) – the primary key for the associated sandman.model.Resource
Return type:

flask.Response
